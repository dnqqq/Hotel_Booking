# Hotel_Booking

### Дисклеймер: гарантований +вайбік
## 1. Назва застосунку

Roomy (або Rozkymar)
## 2. Мета

Застосунок допомагає користувачам швидко та зручно бронювати номери в готелях. Він надає актуальну інформацію про доступні номери та ціни.
## 3. Користувачі та їх можливості

Гості (користувачі, які бронюють житло):

Пошук готелів за фільтрами (ціна, розташування) Перегляд деталей готелю (фото, опис) Бронювання номера Оплата (натурою) Управління бронюваннями (скасування, зміна дат)
Адміністратори:

Підтримка користувачів(моральна) Контроль контенту
## 4. Основні сутності (основні об'єкти, якими оперує застосунок, та їх поля)

### Готель
- **Назва**  
- **Локація**  
- **Опис**  
- **Фото**  
- **Доступні номери**  

### Номер:
- **Тип номера**: стандарт, люкс тощо  
- **Кількість гостей**  
- **Ціна за ніч**  
- **Доступність**: легка  
- **Зручності**: туалет не на вулиці  

### Бронювання:
- **Гість**  
- **Готель**  
- **Номер**  
- **Дата заїзду/виїзду**  
- **Статус**: активне, скасоване, підтверджене, в активному пошуку  

### Користувачі:
- **Ім'я**  
- **Email**  
- **Тип користувача**: гість, адміністратор
